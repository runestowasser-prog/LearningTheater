## 🤔 Why Learning Theater?

If you've ever tried to push tools like Storyline or Rise beyond their intended use, you know the feeling:

- Workarounds
- Timeline limitations
- Rigid interaction models

Learning Theater removes those constraints.

You get:

- Full control over behavior and logic  
- Smooth, flexible animations with **GSAP timelines**  
- A visual editor *and* real code access  
- A powerful **Event System** for interactions  
- No forced workflows or hidden limitations

## 🧠 How it works

Learning Theater is built around a few simple concepts:

- **Actors** — visual elements (text, images, containers)  
- **Properties** — position, size, behavior  
- **Scripts** — `General`, `Build`, `SceneStart`, `Resize`  
- **Event System** — events, conditions, actions  
- **GSAP timelines** — smooth animation control  

Everything updates live while you work.
