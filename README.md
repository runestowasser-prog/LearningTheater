# 🎭 Learning Theater

**Learning Theater** is a visual JavaScript tool for building interactive scenes, simulations, and e-learning — fast, lightweight, and without complex setup.

Design your scene with objects (*Actors*), add behavior visually or with code, build animations with **GSAP timelines**, and create interactivity through the built-in **Event System**.

## ✨ Features

- Build interactive scenes visually
- Work with objects (*Actors*) and hierarchies
- Edit with live preview
- Add logic through the built-in **Event System**
- Create animations with **GSAP timelines**
- Combine GUI editing with direct code access
- Export clean, readable project code
- Run locally without build tools

---

## 🧠 Core concepts

Learning Theater is built around a few simple ideas:

- **Actors** — visual elements such as text, images, and containers
- **Properties** — position, size, appearance, and behavior
- **Scripts** — `General`, `Build`, `SceneStart`, and `Resize`
- **Event System** — create interactions using events, conditions, and actions
- **GSAP timelines** — build smooth, sequenced animations

This makes it possible to work both visually and programmatically in the same project.
